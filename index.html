⚠ One-time setup required. Follow these steps to connect Google Sheets as your database.
1
Create a new Google Sheet
Go to sheets.google.com → create a new blank spreadsheet → name it 99acres Portal DB
2
Open Apps Script
In the sheet: Extensions → Apps Script. Delete any existing code. Paste the entire contents of the Code.gs file. Click Save (💾).
3
Deploy as Web App
Click Deploy → New Deployment → Type: Web App → Execute as: Me → Who has access: Anyone → Click Deploy → Authorize when prompted.
4
Paste your Web App URL below
After deploying, copy the Web App URL and paste it here.
